## Agent skills

### Issue tracker

Issues are tracked locally as markdown files under `.scratch/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Default five-state vocabulary (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` at the repo root, ADRs under `docs/adr/`. See `docs/agents/domain.md`.
