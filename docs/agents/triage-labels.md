# Triage labels

Five canonical statuses used in the `status:` frontmatter field of `.scratch/*/issue.md` files.

| Role                                      | String            |
|-------------------------------------------|-------------------|
| Maintainer needs to evaluate              | `needs-triage`    |
| Waiting on more info from reporter        | `needs-info`      |
| Fully specified, an agent can pick it up  | `ready-for-agent` |
| Needs a human to implement                | `ready-for-human` |
| Will not be actioned                      | `wontfix`         |
